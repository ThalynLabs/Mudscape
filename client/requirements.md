## Packages
react-virtuoso | Efficient virtualized scrolling for the terminal output (handles variable height lines)
anser | Low-level ANSI escape code parser for coloring MUD output
framer-motion | Smooth animations for UI elements and page transitions
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes

## Notes
WebSocket connection required at /ws endpoint
The app mimics a terminal: uses a monospace font and high contrast colors
Accessibility is a priority: uses semantic HTML and aria-live regions for screen readers
